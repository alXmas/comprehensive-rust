# Comprehensive Rust — мои упражнения

Здесь хранятся мои решения задач из курса [Comprehensive Rust](https://google.github.io/comprehensive-rust/) от Google.

## О курсе

Comprehensive Rust — это четырёхдневный курс по языку Rust, разработанный командой Android в Google. Курс охватывает все основные аспекты языка: от базовых типов и управления памятью до многопоточности и unsafe Rust.
